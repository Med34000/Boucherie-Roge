# Boucherie Rogé - Site Vitrine

Site web vitrine pour la Boucherie Rogé à Paulhan (34).
Développé avec Next.js (IAtelier OS Standard).

## Fonctionnalités
- Page d'accueil avec présentation
- Page À Propos (Histoire de Thierry Rogé)
- Page Contact (Plan, Horaires)
- Design Responsive et Premium

## Stack Technique
- Next.js 15
- React
- CSS Modules (Design sur mesure)
- Font: Playfair Display & Inter

## Installation
```bash
npm install
npm run dev
```

## Déploiement
Le projet est prêt à être déployé sur Vercel, Netlify ou Infomaniak (Node.js).
